# React TypeScript Starter Template

A React + TypeScript + TailwindCSS starter template using Vite with some cool pre-configured features, it's fast.

## Features

- Routing with react-route v6

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# build
$ npm run build

# preview
$ npm run preview
```

## Test

```bash
# unit tests
$ npm test
```
